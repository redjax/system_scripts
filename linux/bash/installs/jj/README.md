# Jj - Jujitsu Git

[Jujitsu (jj)](https://github.com/jj-vcs/jj) is a modern take on version control systems like Git and Mercurial. It is currently an experimental system, but because of the way it functions, you can start using it without risking your current codebase. Jujitsu abstracts a lot of complex Git operations, but does not replace them. Operations take in Jujitsu are compatible with your current Git repositories.
