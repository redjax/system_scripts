# Autorestic <!-- omit in toc -->

[Autorestic](https://github.com/cupcakearmy/autorestic) is a configuration-driven backup cli for restic. It works by defining your backups in a central `.autorestic.config`, or with individual `.autorestic.yml` files in paths on your machine.
